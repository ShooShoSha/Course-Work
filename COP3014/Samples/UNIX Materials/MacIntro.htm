Using the Macintosh

 

Remember:

            The Macintosh runs the UNIX operating system.

            At the command line of the Macintosh you will use the “Bash” shell.

            It is best to store your work for this course in one of two places:

                        1. On the CS department's NFS server.

                        2. On the “H:” drive which is managed by ITS.

            Log on to the machine using your Argo username and password.

 

Start with the desktop:

You should see a bar across the bottom of the screen called the “Dock”.  (Similar to task bar.)

 

There will always be a menu bar across the top of the screen. It changes to the current application.

 

On the Dock, find the icon that looks like a compass. That’s a program called “Safari”. That is the Macintosh browser. (FireFox should also be available.) Click and open it. Now notice the menu bar across the top of the screen. It automatically changes to present the menu for the current application.

Play with the buttons at the top left corner (red, yellow, green) of the window. What do they do? Notice that when you minimize the window (yellow button) the minimized document goes to the dock. When you close the window (red button) the document is not on the dock, but the application is still running. To exit the application you go to the “Safari” menu in the menu bar at the top of the screen and choose “Exit Safari”.

 

Now check out the menu at the very top-left corner of the screen that looks like an apple. It is always present regardless of the current application. The two most important items on the menu for you now are “log out” and “system preferences”. It is critical that you always log out before you leave the lab. If you forget a “mean" student could destroy all your files.

 

Check out the system preferences menu. You can experiment as you please with the options, but you might want to start by adjusting the mouse settings. The default speed of the mouse is very slow. You might want to change it. Notice also that the functions associated with the mouse can also be modified. By default, the mouse is a “one button” mouse. If you like to use a “right” click (a two button mouse), you can make that change.  Change the right button to “Secondary Button”. You can make other changes if you like.

 

 

The File System:

On the Dock you will find a program named "Finder". Open a finder window. That is a window into the storage devices available on this machine.

On the desktop you may see icons representing disk drives. If so, open the icon for the hard drive on the local machine.  (perhaps named Macintosh HD)

 

Play with the menu down the left side of the Finder window. Be sure to look in the applications folder. Notice that at the bottom of the applications folder is a Utilities folder. Those two folders let you know what software is installed.

 

Create a folder for this class named “Cot3011”.

Inside that folder create a folder called “Proj1”.

 

During this course you will want to copy documents from the instructor’s web site. Can you copy this document to the new folder you just created?

 

Suppose you go home tonight and want to continue working on a program you started in class. Can you access the NFS server or the H:\ drive from home? If you start a program on your computer at home, can you save a copy on the servers at UWF?

 

The Command Line (Terminal Window):

There is an application on the Macintosh called “Terminal”. It may already be on the Dock. It looks like a computer monitor with a black screen. If it’s not on the dock you will find it in the applications/utilities folder of the hard drive. Start the terminal application. Notice that you can start multiple copies simultaneously.

 

You have now seen the basics of the Macintosh. That is most of what you need to know about the Macintosh for this course. If you want to learn more that is fine, but not necessary.
